# Requirements: KingdeeKB v0.2 — 智能文档生成

**Defined:** 2026-05-23
**Core Value:** 基于实施方法论模板自动生成标准化交付物

## v0.2 Requirements

### Template Engine (TMPL)

- [ ] **TMPL-01**: 系统能解析 .docx 模板的字段占位符（如 `{项目名称}`、`{调研日期}`）和章节结构
- [ ] **TMPL-02**: 系统能解析 .xlsx 模板的工作表结构和字段
- [ ] **TMPL-03**: 模板按金蝶实施方法论 8 阶段分类展示（项目管理→验收）
- [ ] **TMPL-04**: 支持用户上传自定义模板
- [ ] **TMPL-05**: 模板字段元数据定义（字段名、类型、必填/可选、默认值、说明）

### Document Generation (DGEN)

- [ ] **DGEN-01**: 用户选择模板后进入向导式填写流程
- [ ] **DGEN-02**: 系统自动识别模板中需要填充的字段并逐一引导
- [ ] **DGEN-03**: 缺失必填字段时主动向用户追问
- [ ] **DGEN-04**: LLM 根据用户输入内容填充文本字段
- [ ] **DGEN-05**: 系统从知识库检索相关历史内容辅助填充（智能补全）
- [ ] **DGEN-06**: 支持生成 .docx 格式产物（保留原模板样式）
- [ ] **DGEN-07**: 支持生成 .xlsx 格式产物
- [ ] **DGEN-08**: 产物本地存储并可导出

### Key Deliverables (DLVR)

- [ ] **DLVR-01**: 调研报告生成 — 用户提供调研记录 → 输出调研报告.docx
- [ ] **DLVR-02**: 周报/月报生成 — 用户提供工作内容 → 输出周报.xlsx
- [ ] **DLVR-03**: 业务蓝图生成 — 用户提供方案内容 → 输出业务蓝图.docx
- [ ] **DLVR-04**: PCR 生成 — 用户描述变更需求 → 输出需求变更申请单.docx
- [ ] **DLVR-05**: 上线单生成 — 用户提供上线信息 → 输出上线确认单.docx
- [ ] **DLVR-06**: 验收单生成 — 用户提供验收信息 → 输出验收确认单.docx
- [ ] **DLVR-07**: 会议纪要生成 — 用户提供会议记录 → 输出会议纪要.docx

### Product Management (PROD)

- [ ] **PROD-01**: 产物历史列表（按时间、项目筛选）
- [ ] **PROD-02**: 产物预览（渲染 .docx 内容预览）
- [ ] **PROD-03**: 产物重新生成（修改输入后重新填充）
- [ ] **PROD-04**: 产物导出到指定目录

### Frontend UI (FUI)

- [ ] **FUI-01**: 模板选择界面（按阶段分类卡片）
- [ ] **FUI-02**: 向导式填写流程（步骤指示器 + 字段输入）
- [ ] **FUI-03**: 智能补全建议展示（从知识库检索的相关内容）
- [ ] **FUI-04**: 产物预览与编辑界面
- [ ] **FUI-05**: 产物管理界面（历史列表 + 操作）

## Out of Scope (v0.2)

| Feature | Reason |
|---------|--------|
| PPTX 模板渲染 | v0.2 仅支持 .docx + .xlsx，演示文稿 v0.3 |
| 多人协作编辑产物 | 单人使用定位 |
| 在线模板市场 | 社区贡献模板 v0.3 |
| 产物自动上传到项目管理系统 | 远期规划 |
| 模板可视化编辑器 | v0.2 仅支持字段填充，不支持模板编辑 |
