# KingdeeKB 使用指南

金蝶 ERP 实施知识库 — AI 驱动的调研助手、风险把控，为金蝶 ERP 顾问打造的一站式桌面工具。

## 快速开始

### 1. 安装应用

从 [GitHub Releases](https://github.com/neal-kingdee/KingdeeKB/releases) 下载对应平台的安装包：

| 平台 | 安装包格式 |
|------|-----------|
| Windows | `.exe` 或 `.msi` |
| macOS (Apple Silicon) | `.dmg` (ARM) |
| macOS (Intel) | `.dmg` (x64) |
| Linux | `.AppImage` 或 `.deb` |

### 2. 配置 LLM

首次打开应用后，进入「设置」页面：

1. 选择 LLM 服务商（OpenAI / Anthropic / 本地模型）
2. 填写 API Key
3. 设置 API 地址（如使用代理或私有部署）
4. 点击「测试连接」验证配置

### 3. 导入知识库

进入「知识库」页面，点击「导入文档」：

- **支持格式**：Word (.docx)、Excel (.xlsx)、PDF、纯文本 (.txt)、Markdown (.md)
- **批量导入**：选择整个文件夹，自动扫描并导入所有支持的文件
- **项目隔离**：导入时可指定项目名称，实现多项目知识隔离

### 4. 开始对话

进入「AI 助手」页面，输入问题即可开始对话。AI 会自动搜索知识库并给出专业回答。

## 核心功能

### AI 助手（Chat）

核心对话界面，支持：

- **知识库问答**：基于导入的文档回答实施问题
- **多轮对话**：自动维护上下文，支持连续追问
- **附件支持**：粘贴图片或添加文件作为参考
- **流式输出**：实时显示 AI 思考过程和工具调用
- **澄清提问**：AI 不确定时会主动提问澄清

**快捷操作**：
- `Enter` 发送消息
- `Shift + Enter` 换行
- 点击「清空对话」重新开始

### 调研助手（Research Assistant）

结构化的调研记录管理：

1. **创建调研会话**：选择模块、填写访谈对象和日期
2. **逐条记录问答**：按问题-回答对记录调研内容
3. **AI 辅助回答**：基于知识库自动生成参考答案
4. **语音输入**：支持录音转文字（需配置 Whisper）
5. **蓝图提炼**：一键从调研记录中提炼 As-Is / To-Be / Gap

### 风险把控舱（Risk Control）

双轨风险管控工具：

- **需求蔓延警报**：新需求 vs 合同范围的自动比对
- **项目健康度**：4 维度评分（缺席率/数据延迟/问题积压/配合度）
- **防身话术库**：针对常见困境的沟通话术生成
- **AI 深度分析**：与 AI 对话讨论风险策略
- **备份恢复**：项目数据的导出和导入

### 文档生成（Templates）

智能文档生成引擎：

- **7 种白名单模板**：调研报告、蓝图、会议纪要、周报、PCR、上线、验收
- **智能填充**：AI 自动填写模板中的字段
- **知识库引用**：生成内容自动引用知识库中的相关段落
- **批量导出**：一键导出所有已生成的文档

### 技能系统（Skills）

可扩展的外部技能框架：

- **内置技能**：25+ 个开箱即用的技能
- **自定义技能**：支持导入 SKILL.md 格式的技能包
- **脚本执行**：技能可包含 .js/.py/.sh 脚本
- **权限管理**：脚本执行前需用户授权

## 常见问题

### Q: AI 回答不准确怎么办？

1. 确认知识库中有相关文档
2. 尝试换个角度提问
3. 在「设置」中调整 temperature 参数（越低越精确）
4. 使用澄清功能引导 AI 理解你的意图

### Q: 导入文档后搜索不到内容？

1. 检查文档是否在「知识库」页面显示
2. 尝试用关键词而非完整句子搜索
3. 确认 embedding 模型已加载（设置页面查看状态）

### Q: 应用卡住了怎么办？

1. 等待 10 分钟 — Agent 会话有自动超时保护
2. 点击输入框旁的取消按钮中断当前操作
3. 如仍无响应，强制关闭并重新打开

### Q: 如何更新应用？

1. 应用会自动检查更新（如有配置）
2. 手动更新：从 GitHub Releases 下载最新版本安装

## 技术栈

| 组件 | 技术 |
|------|------|
| 桌面框架 | Tauri v2 (Rust + WebView) |
| 前端 | React + TypeScript + Tailwind CSS |
| 向量模型 | bge-small-zh-v1.5 (本地 ONNX) |
| 搜索引擎 | 混合搜索（向量 + BM25 + RRFR 融合） |
| Agent 引擎 | rig-core (Rust) |
| 数据存储 | SQLite + tantivy |
| 语音识别 | Whisper (本地) / 腾讯云 / 讯飞 (在线) |
